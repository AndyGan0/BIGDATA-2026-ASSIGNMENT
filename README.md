# BIGDATA-2026-ASSIGNMENT

This github repository contains the assignment for the course "Large-Scale Data Management".
The enviroment has to be set up based on the guides of the lab.
https://github.com/ikons/bigdata-dsml

## Source code

The source code is in the folder src

* Sub-folder "Parquet_code" contains the necessary files for the experiments that compared the execution time of query 1 with parquet.

* Sub-folder "join_strategy_experiments" contains the necessary files for running queries 3 and 4 while forcing different join strategies.


## Howtos

The folder "howtos" contains guides to run those files in the cluster.


## Results

The folder "results" contains the results of the executions, including:
* outputs
* execution time 
* physical plan (wherever needed)
